## Event App

This is a mobile app...

### Prerequisites

- yarn
- node version 12 or higher
- expo-cli
- A mobile emulator or mobile device

## Getting Started

- Clone the project in your local machine;
- cd into the project directory;
- run `yarn install`
- run `yarn start`
- Open the expo-client app in your android device
- scan the QR code

## Built With
